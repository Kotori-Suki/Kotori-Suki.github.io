# C++11 新特性 使用篇 1

主要内容：不定参数模板偏特化，forward，move，initializer_list<> 的使用

# 不定参数模板偏特化

简介：跟函数重载类似，有不定参数的模板函数，还有一些特化的模板函数，C++ 会根据运行时不定参数的类型来选用不同函数。需要注意的是，没有参数也是一种不定参数，所以如果递归调用一个只有不定参数的函数，是会进入死循环的

# move 和 forward

简介：move 是将左值转换为右值的关键字，常用于移动构造，较少复制次数，提高代码效率。forward 则是为了在模板中使用右值而出现的，当我们在模板中使用右值时，推导出来的具体数据是无法再转换为该模板类型的右值的，forward 就可以很好地解决这个问题

另外，ue 中，forward 关键字被包装成 Forward，而 move 则包装成了 MoveTemp，用法相同。

# initializer_list<>

简介：就是一个模板容器，支持 { } 来初始化其值
