# 算法总结：双指针

概述：双指针是非常有用且基础的算法技巧，可分为快慢指针，左右指针和循环哨兵。思路基本都差不多，这里只列出一些题目，有需要可以去力扣刷

# 快慢指针

​ 快慢指针，即两个指针以不同速度从同一方向开始移动。

解决问题：

  • 环形链表找循环起始点
  • 旋转链表
  • 比较版本号

# 左右指针

​ 左右指针，即两个指针从不同方向开始移动。

解决问题：

  • 三数之和
  • 盛最多水的容器
  • 下一个排列

# 循环哨兵

​ 循环哨兵，即循环时更新数组当前最大 / 最小值，可以将循环和哨兵都理解为指针

解决问题：

  • 买股票的最佳时机 1

# KMP 算法

​ KMP 算法，即先维护匹配串的前缀表，再在原串中进行匹配和跳转

解决问题：

  • 实现 strStr

结语：本章只是为了唤醒我对双指针技巧的记忆，有空可以脑补一下做过的双指针的题。
