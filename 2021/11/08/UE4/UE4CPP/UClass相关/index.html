# UClass 相关

简介：本篇将对 "UClass" 的构造和初始化进行解析，同时也会涉及到 "UClass" 中的所有属性和函数的解析。为什么会想到去研究 "UClass" 呢？主要是因为源码中很多函数的实现，都用到了 "UClass" 中的反射信息。如果想深入地了解这个函数，那就得先了解这个 "UClass" 里面都有些什么属性，其又是怎么工作的。让我们开始吧 😃

前置知识：

  1. 了解 "UE" 的类型系统
  2. 了解 "UHT" 的编译过程
  3. 了解 "SparseClassData" 结构体
  4. 知道 "RPC" 函数的概念
  5. 知道 "反射机制" 的概念

# UClass 简介

首先，"UClass" 是描述一个类的类，里面存的都是被描述的类的一些反射信息。以下是 "UClass" 的继承关系图（大钊的），我们只对 "UClass" 进行展开，当然也会涉及到 "UStruct" , "UField" 和 "UObject" 的解析。其他分支其实也类似，留给读者自己探究。
